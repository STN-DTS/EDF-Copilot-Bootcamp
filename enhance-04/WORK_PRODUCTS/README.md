Each agent should create their work log using this template:

```markdown

**Owned paths:**
- (list files/folders owned)

**Out of scope:**
- (list explicitly excluded)

## Recon summary
- **Files discovered:** (count and list)
- **Issues found:** (broken links, missing content, etc.)
- **Plan:** (5-10 bullet summary)

## Changes made
- **Files created:** (list with line counts)
- **Files modified:** (list with change summary)
- **Total insertions/deletions:** +X / -Y

## Verification
**Commands run:**
```
(paste commands and output)
```

**Results:** PASS / FAIL

## Follow-ups / Risks
- (any issues for Coordinator attention)
- (any dependencies on other agents)
```
