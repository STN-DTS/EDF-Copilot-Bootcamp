# Requirements

Complete before Day 1.

## Checklist

- [ ] GitHub Enterprise account active
- [ ] Copilot license activated
- [ ] `git --version` → 2.30+
- [ ] Copilot completions working
- [ ] Copilot Chat responding
- [ ] Stack tools installed (after selection)

---

## Mandatory

| Requirement | Verify |
|-------------|--------|
| GitHub Enterprise account | Log in at github.com |
| GitHub Copilot Enterprise license | github.com/settings/copilot → "Active" |
| Git 2.30+ | `git --version` |
| IDE with Copilot Chat | VS Code, JetBrains, Visual Studio, or Neovim |

```bash
# Run in terminal
git --version
```

---

## Verify Copilot

1. Open a code file
2. Type a comment → ghost text appears
3. Open Copilot Chat → ask "What is 2+2?" → response received

```bash
# VS Code: Check Copilot extension installed
code --list-extensions | grep -i copilot
```
