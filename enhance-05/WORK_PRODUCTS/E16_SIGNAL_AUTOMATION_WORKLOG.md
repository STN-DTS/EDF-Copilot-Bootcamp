## Mission

Create a signal automation helper script that agents can call to create completion signals uniformly.

## Deliverables

### 1. Signal Helper Script

**File:** `scripts/signal-helper.mjs`

A comprehensive CLI tool for managing multi-agent completion signals.

### 2. Commands Implemented

| Command | Description | Exit Code |
|---------|-------------|-----------|
| `create <ID>` | Create a completion signal | 0=success, 1=error |
| `check <ID>` | Check if signal exists | 0=exists, 1=not found |
| `list` | List all signals with status | 0=success |
| `wait <ID>` | Poll until signal appears | 0=found, 1=timeout |

### 3. Options Supported

| Option | Short | Description |
|--------|-------|-------------|
| `--message` | `-m` | Custom message for signal content |
| `--status` | `-s` | Status value (default: COMPLETE) |
| `--force` | `-f` | Overwrite existing signal |
| `--dry-run` | `-d` | Preview without creating file |
| `--timeout` | `-t` | Timeout for wait command (seconds) |
| `--interval` | `-i` | Poll interval for wait (seconds) |
| `--help` | `-h` | Show help message |

## Usage Examples

```bash
# Create a signal
node scripts/signal-helper.mjs create E17 --message "Task completed"

# Check if another agent is done
node scripts/signal-helper.mjs check E15

# List all signals
node scripts/signal-helper.mjs list

# Wait for a dependency with timeout
node scripts/signal-helper.mjs wait E15 --timeout 30

# Dry run to preview
node scripts/signal-helper.mjs create E18 --dry-run
```
